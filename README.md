# Suspicious

Detects possibly suspicious stuff in your source files

## Installing this

### from source

```bash
git clone git@github.com:sturdy-dev/suspicious.git
cd suspicious
python -m pip install .
```

## Running this

```bash
sus /path/to/file.py
```
